Populate three HTML files with the specified elements and content. Styling is optional.

1-1: signup.html

  1. Valid and file created. The HTML must be valid according to html-validate.
  2. A form element is present with 2 input elements: one of type email for email, and one input of type ‘checkbox’, and a required submit button with the text “Submit”.
  3. Add a password input of type “password”
  4. Each input element must be wrapped in a paragraph tag.
  5. Each input element should have a corresponding label, which is associated **explicitly* (using the for attribute) with the ids: “email”, “signup”, “passowrd”. The label for the checkbox should be exactly “Sign up for our newsletter”. Do not include the input as a child of the label in this case.
  6. The checkbox is set as default checked
  7. Both email and password must be required.
  8. The form action should send the results to “thanks.html”

1-2: index.html

  4. Each article should have the class “post”
  5. The posts should contain both a h2 element and one or more paragraph elements, and not be empty. The content is up to you.
